# gobang-js

A gobang game server.

![preview](https://github.com/lusterofgem/gobang-js/blob/main/assets/images/preview.png)

The client & serveer use ws to transfer message to communicate.

The message is a json (stringifed object), the structure is like this:

```
{
    "type": "chat",
    "content": "hello there"
}
```

## using port

 - express: 8080

 - ws: 8081

## client to server message
 
 - put-chess

```
{
    "type": "put-chess",
    "content": "0,0"
}
```

 - restart-game

```
{
    "type": "restart-game",
}
```

 - chat

```
{
    "type": "chat",
    "content": "hello there"
}
```

## server to client message

 - put-chess

```
{
    "type": "put-chess",
    "content": "0,0,black"
}
```

 - sync-checkerboard

 ```
 {
    "type": "sync-checkerboard",
    "content": [
        ["", "", "", ...],
        ["", "", "", ...],
        ["", "", "", ...],
        ...
    ]
 }
 ```

 - sync-winner
 
```
{
    "type": "sync-winner",
    "content": "black"
}
```

 - sync-turn

```
{
    "type": "sync-turn",
```
